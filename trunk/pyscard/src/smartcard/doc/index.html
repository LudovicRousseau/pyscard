pyscard Documentation

Last update : pyscard 1.6.0 (May 2007)

pyscard is a python module adding smart cards support to python

It consists of smartcard.scard, an extension module wrapping Windows smart card base components (also known as PCSC) on Windows and PCSC lite on linux, and smartcard, a python framework hiding PCSC complexity.
